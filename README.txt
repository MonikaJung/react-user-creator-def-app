Simple React app to manage users.

Project goal: practice work with portals, fragments and refs.

To make the app work:
1. Clone the repository
2. Install Node.js
3. Open the project (VS Code or your fav editor)
4. Open local terminal (in the repository location .../React-user-creator-def-app>)
5. Run "npm install" command and wait
6. Run "npm start" command and enjoy the app

Project created with this course:
